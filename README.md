# EAN-13 条形码生成器

一个简单易用的网页应用，用于生成EAN-13条形码。

## 使用方法

### 方法一：直接打开（最简单）

1. 找到项目文件夹中的 `index.html` 文件
2. **双击** `index.html` 文件
3. 文件会在您的默认浏览器中打开

### 方法二：使用本地服务器（推荐）

如果直接打开遇到问题（比如CDN资源加载失败），可以使用本地服务器：

#### Windows用户：

1. **双击运行** `启动服务器.bat` 文件
2. 等待服务器启动
3. 在浏览器中访问：**http://localhost:8000**

#### 或者使用PowerShell：

1. 右键点击 `启动服务器.ps1`
2. 选择"使用PowerShell运行"
3. 在浏览器中访问：**http://localhost:8000**

### 方法三：手动启动服务器

如果您安装了Python：

```bash
python -m http.server 8000
```

然后在浏览器中访问：**http://localhost:8000**

## 使用说明

1. 在输入框中输入**3位数字**（例如：690）
2. 点击"生成条形码"按钮
3. 查看生成的13位EAN码和条形码图片

## 功能特点

- ✅ 自动生成9位产品代码
- ✅ 自动计算校验位
- ✅ 生成可扫描的条形码图片
- ✅ 美观的现代化界面
- ✅ 响应式设计，支持移动设备

## 文件说明

- `index.html` - 主页面
- `style.css` - 样式文件
- `app.js` - 核心功能逻辑
- `启动服务器.bat` - Windows批处理启动脚本
- `启动服务器.ps1` - PowerShell启动脚本

## 注意事项

- 确保网络连接正常（需要加载CDN上的JsBarcode库）
- 如果直接打开HTML文件遇到问题，建议使用本地服务器方式
